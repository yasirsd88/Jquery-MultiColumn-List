jQuery Multi Column List
=============
**jQuery Multi Column List ** is a jQuery plugin that provides option to divide fields into equall columns.

```javascript
// USAGE.

$('.multilist-container').multiColumn({minfields: 10,columns: 2});

 ```
 ```html
<div class="multilist-container">
	<fieldset>Elements goes here.</fieldset>
	<fieldset>Elements goes here.</fieldset>
	<fieldset>Elements goes here.</fieldset>
</div>

// CSS
.multi-columns{float:left}
 ```
